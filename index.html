    The scope of a var variable is functional scope. The scope of a let variable is block scope. The scope of a const variable is block scope.It can be updated and re-declared into the scope.     Differences of arrow function and regular function are given below:

    The first and most obvious difference between arrow functions and regular functions is their syntax.Inside a regular function, execution context (i.e. the value of this) is dynamic.On the other hand, an arrow function does not define its own execution context. Regular functions can be used as constructors, using the new keyword. Yet another consequence of the lexical resolution of this inside arrow functions is that they cannot be used as constructors
